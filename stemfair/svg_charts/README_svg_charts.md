# SVG Charts (Poster-ready)

- `city_warming_anomaly_comparison.svg` — Compares Elko, Reno, and Las Vegas warming using **temperature anomalies** (baseline 1975–2000). Anomalies make the comparison fair because the cities have different normal temperatures.

- `elko_energy_needs_change.svg` — Shows how Elko’s **energy-related climate indicators** changed (2001–2025 vs 1975–2000). Heating = HDD65, Cooling = hot days (TMAX ≥ 90°F), plus a combined index.

- `elko_snowfall_time_travel.svg` — Compares Elko winter snowfall across two **aligned 25-year eras** (1975–2000 vs 2001–2025) to make the distributions easier to compare.

- `DJF_avg_temperature_trend.svg` — Elko winter (DJF: Dec–Feb) average temperature trend over time.

- `freeze_days_trend.svg` — Elko winter freeze-day trend (whether winters have more or fewer freezing days).

- `winter_HDD65_trend.svg` — Elko winter heating demand trend using HDD65 (higher HDD usually means a colder winter and more heating needed).

- `winter_total_snowfall_combo.svg` — Elko winter total snowfall summary/trend chart.
